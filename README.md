# Nimble Dashboard Instructions

Reusable WordPress admin plugin by Nimble.Help.

## Features

- Dashboard instruction banner.
- Media Library image optimization warning.
- Media upload modal image optimization warning.
- Admin menu highlights for Dashboard, Posts, Media, and Pages.

## Update Setup

This plugin includes Git Updater headers:

- `GitHub Plugin URI: https://github.com/Ljuk67/Nimble-Dashboard-instructions`
- `Primary Branch: main`

Install [Git Updater](https://github.com/afragen/git-updater) on client sites to receive updates from this GitHub repository.
