Adds reusable Nimble.Help dashboard instructions and image-optimization reminders in the media library and upload modal.

== Description ==

Nimble Dashboard Instructions adds simple admin-facing reminders for editors and content managers:

* a permanent, full-width instruction section at the top of the main dashboard
* a prominent image-optimization warning in the media library
* the same image-optimization warning in the WordPress media upload modal
* highlighted Dashboard, Posts, Media, and Pages admin menu items using the Nimble.Help accent colour

The dashboard instruction list can be customized with the `nimble_dashboard_instructions_items` filter.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress admin.
3. Open Dashboard to review the instruction banner.
